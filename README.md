## Jashore Foodies
This Project is created with React.js, Express,js, Firebase and React Router, 
## Technology Used
* React.js
* Firebase
* Express.js
* MongoDB
* React Router
* Axios
* TanStack Query
* React Swiper
* TailwindCSS
* React Private Route
* React Toastify
